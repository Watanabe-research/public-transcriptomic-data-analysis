# public-transcriptomic-data-analysis

# Repository overview
Input data are stored in the data folder. The sources of each input file are listed below.
Output data generated by the scripts are stored in the output folder.

# Input data
•	patients_and_t_log2_mRNA_data

Processed microarray data obtained from GSE20181:
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE20181

•	gene_name
Probe annotation file containing probe IDs, gene symbols, and Entrez IDs obtained from GPL96:
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GPL96

•	h.all.v2024.1.Hs.entrez
HALLMARK gene sets obtained from MSigDB release 2024.1:
https://data.broadinstitute.org/gsea-msigdb/msigdb/release/2024.1.Hs/

•	after_probe_selection_gene_expression_data
Gene expression matrix generated after probe selection from patients_and_t_log2_mRNA_data.

•	after_probe_selection_gene_expression_data_processed
Processed dataset containing the subset of data required for downstream analyses.

•	patients_Cluster_information
Cluster assignments and associated sample information generated by hierarchical clustering.

•	INFLAMMATORY_cluster_labels
Cluster labels extracted from patients_Cluster_information.

•	HALLMARK_INFLAMMATORY
Gene symbols and Entrez IDs included in the HALLMARK_INFLAMMATORY_RESPONSE gene set.

•	all_cells.rds
Seurat object containing all cells used for the scRNA-seq analysis.

# Data availability
The datasets analyzed in this study are publicly available in the Gene Expression Omnibus (GEO) database under accession numbers GSE228499, GSE161529, and GSE20181.

# Software
All analyses were conducted in the R environment.
